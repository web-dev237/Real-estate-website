# Real Estate Landing Page

This is a front-end project where I built a **responsive real estate landing page** using only HTML and CSS.
The goal of the project was to practice layout systems like **CSS Grid**, **Flexbox**, and responsive design.

The page includes a hero section, property listings, testimonials, FAQs, and a footer — similar to what a real property platform homepage would look like.

---

## What I Practiced

* Semantic HTML structure
* CSS Grid for layouts
* Flexbox for alignment
* Responsive design using media queries
* Building reusable UI components (cards, buttons, sections)
